## Introduction:
Hello, my name is Alex, an aspiring game developer. I have always been interested in coding. I used to make games on Roblox, though no one ever really played them. I've always had a passion for atmospheric games. My goal is to create a hardcore, story-based survival game with inspiration from games like Marathon and S.T.A.L.K.E.R. GAMMA.
